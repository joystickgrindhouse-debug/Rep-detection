Overview

Solo Mode is a single-player fitness game where users draw exercise cards and perform the prescribed exercises while real-time pose detection via MediaPipe tracks their form and counts repetitions (reps).

Game Flow

1. Session Start

User clicks "START WORKOUT"
Camera permission is requested
MediaPipe Pose detection initializes
First card is automatically drawn
Workout timer begins
2. Card Draw

Rep counter increments as valid reps are detected

Card Generation

Cards are drawn completely randomly:

Random suit selected (1 of 4 categories)
Random exercise selected from that category (1 of 4)
Random rep goal selected (2-13)
Each draw is independent; exercises can be repeated in sequence.

Pose Detection & Rep Counting

MediaPipe Pose System

Tracks 33 body landmarks (joints, angles)
Runs at real-time frame rate from video stream
Displays skeleton overlay on video (green lines connecting joints)
Red dots indicate tracked landmarks
ExerciseEngine

Analyzes landmark data per exercise
Implements form validation for each exercise type
Counts valid reps based on:
Joint angle thresholds
Movement range validation
Proper form completion
Visibility confidence checks
Example: Push-up Detection

Detects arm bending/extension
Validates shoulder-elbow-wrist alignment
Counts full rep when arms return to extended position
Prevents false counts from partial movements
Stats & Progression

Session Tracking

Current Exercise: Current card's exercise name
Current Reps: Reps completed for current exercise
Rep Goal: Target reps for current card
Real-Time Display

Header shows session totals
Card displays current target
Rep progress bar/counter visible
Session Completion

When user ends session:

Final session statistics calculated
Rewards computed and distributed
Results saved to user profile
Reward System

Dice Calculation

Formula: Math.floor(totalReps / 30)
Example: 95 total reps = 3 dice earned
Dice are game currency for other game modes
Experience Tracking

Total reps contributed to user profile
Dice balance incremented
Stats persisted in Firebase
